# Евгения Калмурзина - Личный ассистент

## Описание проекта
Одностраничное резюме-портфолио для позиции личного ассистента/администратора.

## Основные разделы
- **Заголовок** - имя, должность, контакты
- **Обо мне** - краткая информация, ожидания по зарплате, ключевые показатели
- **Опыт работы** - хронология в виде таймлайна
- **Навыки** - сгруппированы по категориям (управление, организация, коммуникация, технические)
- **Увлечения** - хобби и интересы
- **Контакты** - различные способы связи

## Технические особенности
- Адаптивный дизайн (mobile-first)
- Анимации GSAP при скролле
- Современный UI с градиентами и эффектами
- Оптимизировано для всех устройств
